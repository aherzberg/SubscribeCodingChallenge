# Subscribe Coding Challenge 

## Requirements
- Ruby 3.3+

## Getting Started
2. Install required dependencies
   ```bash
   bundle install
   ```

3. Run the program with a valid text file
   ```bash
   ruby main.rb <input_file.txt>
   ```

Example input file:
```
2 book at 12.49
1 music CD at 14.99
1 chocolate bar at 0.85
```
